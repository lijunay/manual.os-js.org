# OS.js 手册源码

* v3: Uses `gitbook`
* v2: Uses `metalsmith`

Live pages: https://manual.os-js.org/

## Usage

```
// Build v3
npx gitbook build

// Build v2
node index.js
```

You can now serve `_book/` or `_metalsmith/`.
